## 📌 About The Project

ANKITVERSE is a responsive personal portfolio website designed for recruiter review, project showcasing, and professional developer branding.

The goal of this project is to present my skills, selected projects, contact information, and GitHub work in a clean and modern way.

## ✨ Features

- Responsive portfolio design
- Modern dark/futuristic UI
- Project showcase section
- Skills and tech stack section
- Contact and social links
- GitHub Pages deployment
- Mobile-friendly layout
